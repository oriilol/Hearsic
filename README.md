# 🎵 Hearsic 

![Android](https://img.shields.io/badge/Platform-Android-3DDC84?style=flat-square&logo=android)
![Kotlin](https://img.shields.io/badge/Language-Kotlin-7F52FF?style=flat-square&logo=kotlin)
![Compose](https://img.shields.io/badge/UI-Jetpack_Compose-4285F4?style=flat-square)

**Hearsic** is a beautiful, fast, and feature-rich offline local music player for Android. Built from the ground up using modern Android development tools like **Jetpack Compose** and **Media3 (ExoPlayer)**, it brings a premium listening experience directly to your device without the need for an internet connection.


## ✨ Features

* 🎨 **Material You Design:** The app adapts its colors to your device's wallpaper for a seamless, personalized look.
* 🎧 **Background Playback & Notifications:** Fully integrated with Android's native media session. Music keeps playing when the app is closed, with a beautiful notification card.
* 📝 **Custom Lyrics:** Add and save your own lyrics for any song. They are stored locally and will be right there the next time you play the track.
* 🎛️ **Playback Speed Control:** Easily change the speed from `0.5x` (Slowed) to `2.0x` (Nightcore) in real-time.
* 🗂️ **Playlist Management:** Create custom playlists and queue up your favorite songs with a single tap.
* 🔀 **Smart Mix:** Instantly shuffle and play all songs from a specific artist or playlist.
* 🌍 **Multilingual:** Available in 9 languages (English, Spanish, French, German, Italian, Portuguese, Japanese, Russian, and Chinese).
* ☀️ **Dynamic Greetings:** Greets you with "Good morning", "Good afternoon", or "Good evening" based on your local time.

## 🛠️ Tech Stack

This project was built using the recommended modern Android architecture:
* **[Kotlin](https://kotlinlang.org/):** The primary programming language.
* **[Jetpack Compose](https://developer.android.com/jetpack/compose):** For a fully declarative, native UI.
* **[Media3 (ExoPlayer)](https://developer.android.com/guide/topics/media/media3):** For robust, stable audio playback and media session handling.
* **[Coil](https://coil-kt.github.io/coil/compose/):** For fast and efficient album art image loading.

## 🚀 Download & Install

You can download the latest version of Hearsic directly from the Releases page:

1. Go to the [Releases](../../releases) tab.
2. Download the latest `Hearsic.apk` file.
3. Open the file on your Android device and install it (you may need to allow "Install from unknown sources").
4. Enjoy your music! 🎶
